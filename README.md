## 导入预设

Step.1 请在浏览器中打开仓库中的 Lv2_Presets.md 文件，或者点击 [**这里**](https://github.com/RedAsteroid/EurekaOrthos_Presets/blob/main/Lv2_Presets.md)。

```
https://github.com/RedAsteroid/EurekaOrthos_Presets/blob/main/Lv2_Presets.md
```

Step.2 点击文本右端的 `复制快捷键` 浮现提示 `copied!` 即复制成功。

<img src="https://raw.githubusercontent.com/RedAsteroid/EurekaOrthos_Presets/main/img/3.png" width=80% alt=5;/>

Step.3 请打开 `Splatoon` 插件窗口，点击 `布局` 选项卡，点击 `Import from clipboard(从剪切板导入预设数据)` 按钮，即可导入预设。

<img src="https://raw.githubusercontent.com/RedAsteroid/EurekaOrthos_Presets/main/img/4.png" width=80% alt=6;/>

Step.4 全部导入完成后，您可以单人进入正统优雷卡来测试绘制是否生效！
